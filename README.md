# Omer Karatas Resume

A modern, responsive resume website built with HTML and Tailwind CSS.

> [!NOTE]
> **Live Demo:** You can view the live version of the resume here: [theomerkaratas.github.io/resume](https://theomerkaratas.github.io/resume/)

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop screens.
- **Dark Mode**: Supports both light and dark themes with a dedicated toggle.
- **Dynamic Typewriter Effect**: Highlights key roles and skills.
- **Professional Layout**: Clean and modern aesthetic using custom Google Fonts.

## Technologies Used

- HTML5 & CSS3
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- Vanilla JavaScript for interactivity

## How to View

Simply open `index.html` in any modern web browser.
